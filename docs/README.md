# 3D Personality App Docs (v1)
Generated: 2025-12-27 (Asia/Tokyo)

このZIPは、**Big Five/HEXACO＋アタッチメント＋SDT**を統合した性格診断アプリを、**vibe coding（Codex / ClaudeCode / Gemini）**で開発するための関連ドキュメント一式です。

## 目的
- 女子高生・女子大生がハマる「シェアしたくなる診断体験」
- 中身は連続スコア中心（科学寄り）、見せ方はキャラ化（エンタメ寄り）
- Awwwards系の3D/アニメーション/ホバーで“触って気持いい”UI

## 推奨スタック（要件）
- Next.js 16 App Router（3DはClient Component）
- TypeScript / Tailwind / shadcn/ui
- Three.js + React Three Fiber
- モバイルは3D軽量化／フォールバック
