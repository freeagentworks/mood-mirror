# 00 Overview

## プロダクトの一言
Awwwards系の3D/アニメーション体験で「触り心地が良い」診断を提供し、女子高生・女子大生が**友達同士で回したくなる**性格診断アプリ（Web）を作る。

## 研究ベース（採用モデルの位置づけ）
- Trait: Big Five互換の見せ方を保ちつつ、HEXACOの核であるHonesty–Humilityを足した6軸（E/C/O/A/Em/H）。
- 対人: 成人アタッチメント（不安/回避の2軸）。
- 動機: SDT（自律/有能/関係の基本欲求“満たされ度”）。
- 重要: 断定型の決めつけを避け、「いまの傾向」「変化可能」「参考」を明示。

## 成功指標（KPI）
- 完走率（開始→結果表示）: 60%+
- シェア率（結果カード画像/リンク）: 15%+
- 30日再受検率: 20%+
- Web Vitals（目標）: LCP < 2.5s / INP < 200ms / CLS < 0.1

## MVP範囲
- 診断（48問）→スコア→結果（トップ/プロファイル/恋愛/やる気）→シェアカード生成
- アカウント無し（ローカル保存）
- 3D背景（軽量）＋ホバー/スクロール演出（モバイルは簡略化）
