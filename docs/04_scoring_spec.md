# 04 Scoring Spec

## 尺度
Traits: E/C/O/A/Em/H（6軸）
Attachment: Anx/Avd（2軸）
SDT: Aut/Com/Rel（3軸）
合計11スコア（0-100）

## 逆転
(R): 6 - raw

## スコア
尺度=平均(1..5) → (mean-1)/4*100

## Attachment 4象限（閾値60）
Secure / Anxious / Avoidant / Fearful

## SDT派生
needs_mean=(Aut+Com+Rel)/3
needs_bottleneck=min(Aut,Com,Rel)

## 品質チェック
注意問題 / 矛盾検知 → 結果は参考扱い + 再受検導線
